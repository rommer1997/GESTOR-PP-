# Instrucciones para Versión Portable y Escalable

Has descargado el proyecto completo del Gestor de Propuestas Verisure. Este paquete incluye la **Versión Portable** que funciona con un solo clic.

## 1. Versión Portable (Doble Clic y Listo)
Esta es la versión final diseñada para usuarios sin conocimientos técnicos.
- **Ubicación**: Abre la carpeta **`PONLOACA`** y busca el archivo **`VERISURE_APP_PORTABLE.html`**.
- **Cómo usar**: Simplemente haz **doble clic** en el archivo. Se abrirá automáticamente en tu navegador (Chrome, Edge, etc.).
- **Sin Instalación**: No necesitas instalar nada. El archivo contiene todo lo necesario para funcionar por sí solo. Solo necesitas conexión a internet para que la Inteligencia Artificial pueda generar los textos.
- **Rapidez**: Hemos configurado el motor de IA más rápido para que no tengas que esperar.

## 2. Envío por Teams
Si quieres compartir esta herramienta con un compañero por Teams:
1. Envía únicamente el archivo `VERISURE_APP_PORTABLE.html`.
2. Tu compañero solo tendrá que descargarlo y hacer doble clic.

## 3. Integración con PowerApps / SharePoint
- **SharePoint**: Puedes subir el archivo `PONLOACA/VERISURE_APP_PORTABLE.html` a una biblioteca. Puedes renombrarlo a `propuesta.aspx` para que se ejecute como una página interna.
- **PowerApps**: Usa el control "Pantalla de HTML" o un "Iframe" apuntando a la dirección del archivo en SharePoint.

## 4. Automatización con Outlook (Carpeta `integrations/outlook`)
Si quieres que el envío sea automático al terminar la propuesta, revisa la carpeta `integrations/outlook` donde tienes el script listo para conectar con tu cuenta de Microsoft 365.
